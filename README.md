# ❤️ Heart Disease Analysis Dashboard

A *Tableau-based Healthcare Analytics Project*

---

## 📌 Project Overview

*Heart Disease Analysis* is a data visualization project developed using *Tableau* and *Tableau Prep Builder* to analyze patterns and risk factors associated with heart disease.

The dashboard helps healthcare professionals and analysts explore patient data to identify trends, risk indicators, and potential areas for preventive care.

The project transforms raw healthcare data into *interactive dashboards* that support better decision-making and provide insights into how factors such as *age, cholesterol, blood pressure, and other attributes* influence heart disease outcomes.

---

## 🎯 Project Objectives

- Analyze healthcare data related to heart disease  
- Identify key risk factors affecting heart disease  
- Build interactive dashboards for healthcare insights  
- Provide visual storytelling for better understanding of patient data  
- Support data-driven healthcare decision making  

---

## 🛠️ Technologies Used

| Tool | Purpose |
|-----|--------|
| Tableau Desktop | Data visualization and dashboard creation |
| Tableau Prep Builder | Data cleaning and preprocessing |
| Tableau Public | Dashboard publishing and sharing |

---

## 📊 Key Visualizations

The project includes multiple visualizations to answer important healthcare questions:

- Age vs Heart Disease Distribution  
- Cholesterol Level Analysis  
- Resting Blood Pressure Comparison  
- Chest Pain Type Distribution  
- Heart Disease Count by Gender  
- Maximum Heart Rate Analysis  
- ST Depression Impact Visualization  
- Target (Disease Presence) Distribution  
- Overall Heart Disease Risk Insights  

Each visualization helps understand how different factors influence the likelihood of heart disease.

---

## 📂 Project Workflow

The project follows a structured analytics workflow:

### 1️⃣ Data Collection
Heart disease dataset containing patient health attributes.

### 2️⃣ Data Preprocessing
- Cleaning missing values  
- Data formatting  
- Preparing dataset using Tableau Prep Builder  

### 3️⃣ Data Visualization
Creating charts and dashboards in Tableau.

### 4️⃣ Dashboard Development
Combining visualizations into an interactive dashboard.

### 5️⃣ Insight Generation
Identifying patterns and healthcare insights.

---

## 🔍 Key Insights

Some insights derived from the analysis include:

- Heart disease occurrence increases with *age*  
- Certain *chest pain types* are strongly associated with heart disease  
- *High cholesterol levels* correlate with increased heart disease risk  
- *Maximum heart rate achieved* can indicate cardiovascular health  
- *Gender differences* exist in heart disease distribution
